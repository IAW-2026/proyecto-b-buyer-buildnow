# BuildNow Buyer App

Buyer App de BuildNow, una plataforma de delivery de materiales de construcción. La app permite a compradores buscar productos por tienda/categoría/nombre, gestionar carrito, perfil y direcciones, hacer seguimiento de pedidos y acceder a un panel admin simple. También integra la API externa GEOREF para autocompletar ciudades argentinas (direcciones del usuario) y cuenta con un Asistente de materiales con IA que recomienda productos disponibles en una tienda según la necesidad del usuario.
